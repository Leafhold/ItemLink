# ItemLink
A high-performance, standalone Paper plugin that allows players to showcase their currently held item, complete with full NBT tooltips, directly within global chat using interactive placeholders.

## Features
- **Rich tooltips**: Natively bundles item metadata (enchantments, custom display names, lore, potion data, etc.) into a client-readable hover component.
- **Async execution**: Parsing, regex compilation and text replacement run entirely in the background.
